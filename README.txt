-------------------------
Steps to Run the Project
-------------------------


Step 1 -> Open Terminal in the Project Folder Location

Step 2 -> install Environment with (pip install virtualenvwrapper-win) command

Step 3 -> Create a virtualenv with  (mkvirtualenv ProjectDjango) Command

Step 4 -> Install Django in that environment with (pip install django) Command

Step 5 -> (python manage.py runserver) use this command to run the Project

Step 6 -> (http://127.0.0.1:8000/) copy this code and paste in browser

Step 7 -> Hello World Message is Printed in the Webpage.
